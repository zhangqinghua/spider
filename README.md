## HotelMapping对接参数
| Vo属性 | 描述 |
| :-: | :-: |
| name | 接口酒店名称 |
| nameEn |酒店英文名称 |
| startLevel |酒店星级 |
| hotelId |酒店ID |
| country |国家(中国) |
| countryCode |国家代码，可跟country一致(中国) |
| province | 省份(广东省) |
| city | 城市名称(广州) ，没有填上一级 |
| city_code | 城市代码(可跟city一致:广州 ) |
| city_code | 城市代码(可跟city一致:广州 ) |
| address | 酒店详细地址 |
| address_en | 酒店详细地址(英文) |
| phone | 酒店电话 |
| post_code | 邮政编码 |
| images | 图片url，多张用,分割 |
